⭐ Melhor prática (recomendado)
No topo do script adiciona:
const API_URL = "/api/v1";
E depois usa:
fetch(`${API_URL}/notifications`)
fetch(`${API_URL}/notifications/read-all`)
apiFetchJson(`${API_URL}/users/me`)
apiFetchJson(`${API_URL}/users/search?...`)
Assim se mudares versão da API no futuro mudas só 1 linha.
